Intermediate artifacts — for debugging why pairs got the verdicts they did

These files are written to the run directory on every validation. They're intermediate steps — feel free to ignore unless something looks wrong.

  • actual_raw_llm.md — verbatim LLM output from the human-review conversion step, before parsing
  • actual_converted.md — the parsed and normalized human review in AI-review markdown format
  • alignment_llm_analysis.md — verbatim LLM response from the single batch-similarity call (the raw analysis)
  • alignment_similarities.md — N × M similarity matrix parsed from the LLM response, with per-row verdict
  • alignment_ranking.md — human comments sorted by best-match similarity, highest first
